# cv-weapon-tracking
## Integrable weapon recognition system based on a model of a handgun training, trained on its own data set.
Client-side app:
* Python (cv2, darknet, PyQt5, numpy)
* YOLOv4
* Qt 5 Designer
* Google Collaboratory (for automatization)
* labelImg

Server-side app:
* HTML
* CSS
* JavaScript
* NodeJS
* Firebase (NoSQL DB)
